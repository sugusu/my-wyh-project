# 四、TSGS rasterizer 的 alpha 是合成权重，不是 GT 的光学衰减诊断量

我们前面的 Stage 3 已经把 rasterizer 公式查得很清楚：

单 Gaussian：

[
\alpha_i
========

o_i\exp(power_i)
]

再经过 clamp、skip。

多个 Gaussian：

[
T=\prod_i(1-\alpha_i)
]

累计 alpha 本质上对应：

[
1-T
===

1-\prod_i(1-\alpha_i)
]

它回答的是：

> Gaussian footprint（高斯投影核）经过 opacity 加权后，在该像素累计覆盖/合成了多少。

而 GT：

[
A_{gt}
======

1-\exp
\left(
-\frac{1}{3}
\sum_c\tau_c
\right)
]

回答的是：

> RGB 三通道物理消光深度取平均后，对应的标量衰减量。

所以：

[
\boxed{
A_{\rm raster}
\neq
A_{\rm gt}
}
]

一般情况下根本没有理由相等。
