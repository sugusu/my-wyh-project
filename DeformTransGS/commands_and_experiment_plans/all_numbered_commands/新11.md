这个结果我接受：

```text
CASE DEFORMATION-STATE-HIERARCHY-IDENTIFIABLE
```

它证明了一个很重要、但范围有限的事实：

> **(Q_1=J_s)、(Q_2=(\lambda_1,\lambda_2))、(Q_3=C_t) 不是换名字描述同一件事；通过匹配形变对，可以分别制造“Q1 相同但 Q2 不同”以及“Q2 相同但 Q3 不同”的情况。**

因此新主线可以继续。

# 一、N0 真正证明了什么

现在已经有两级关键反例：

### 面积标量不一定够

PAIR-A/B 中存在：

[
J_s^{(a)}=J_s^{(b)}
]

但主拉伸或剪切状态不同。

所以如果某种光学响应在这两个形变下不同，那么只输入 (J_s) 的模型无论多强都无法区分它们。

### 无序主拉伸谱也不一定够

PAIR-C：

[
(\lambda_1,\lambda_2)^{(a)}
===========================

(\lambda_1,\lambda_2)^{(b)}
]

但主拉伸相对于材料 (u/v) 方向的朝向不同，导致：

[
C_t^{(a)}\ne C_t^{(b)}
]

所以如果材料光学响应与材料轴方向有关，只输入无序主拉伸谱也无法区分，必须保留材料坐标系中的完整张量状态。

这就是一个有效的状态层级：

[
J_s
\subset
(\lambda_1,\lambda_2)
\subset
C_t
]

# 二、但当前结果还有三个必须处理的问题

## 1. 条件数极大

结果是：

[
\kappa\approx10^{15}\sim10^{19}
]

这说明描述量矩阵严重共线或量纲失衡。

因此：

> `matrix rank = 10` 不能当作“10 个信息维度都稳定可学习”的证据。

N0 能成立，主要依靠的是**严格构造的 matched pairs（匹配形变对）**，不是相关矩阵或数值秩。

后续不能用普通线性回归系数大小、特征重要性或相关性排名来声称哪个描述量关键。它们会被共线性污染。

## 2. 14 个形变名称中实际存在重复矩阵

至少包括：

* `D0_IDENTITY` = `A1_IDENTITY_JS1`
* `D5_ANISO_X1P60_Y0P80` = `C1_ANISO_X1P6_Y0P8`
* `D6_ROTATION_Z_30` = `D2_RIGID_RZ30`

所以：

```text
candidate deformation count = 14
```

是协议条目数量，不是独立矩阵数量。

否则会出现伪重复。

```text
optical necessity claimed: NO
```

目前只证明：

> 如果存在面积敏感、各向异性敏感或方向敏感的材料机制，我们的形变集合能够把它们区分出来。

还没有证明现实半透明材料一定响应这些状态。

# 三、下一步最危险的做法

现在不能自己随手写：

```text
Mechanism 1: tau ∝ 1/Js
Mechanism 2: tau ∝ lambda1
Mechanism 3: tau ∝ C11 + C12
```

然后比较 Q1/Q2/Q3。

因为这仍然是：

> 用某个 descriptor 生成答案，再发现包含该 descriptor 的输入效果最好。

最多证明实验管线能找回人工规则，不足以支撑论文的主要科学结论。

经典 deformable reflectance field 已经研究形变后的局部方向重参数化和外观保持；DR-GS 也已经把几何、材料与光照解耦并支持形变感知渲染。因此“使用局部 frame 或 (F)”本身不能构成创新。([三菱电机研究院][1])

真正可用的物理边界是：真实受力材料可能出现各向异性、透明度和双折射变化；部分实验还显示剪切或拉伸相关的张量不变量、材料取向以及观测方向可能影响光学响应。([arXiv][2])

但需要特别注意：

> **普通 RGB 强度不一定能观察到纯双折射。**

双折射主要改变光的偏振状态和相位。如果没有偏振片、偏振相机或 Stokes/Mueller（偏振状态/传输矩阵）建模，Q3 的方向信息可能在普通 RGB 透射图中不可见。

所以 D1-N1 首先必须决定：

1. 继续做普通 RGB，那么只能研究**各向异性吸收、消光或散射**；
2. 引入偏振，则可以研究**应变—双折射张量响应**，但工程难度会明显增加；
3. 不允许把双折射机制强塞进普通 RGB Beer–Lambert 渲染器。

# 四、我的真实路线建议

我建议先不进入大型 benchmark，也不训练网络。

下一阶段只做：

> **D1-N1：光学机制与可观测量边界闭环**

它要决定哪一种机制同时满足：

* 有真实物理依据；
* 能被现有图像或可扩展传感器观察；
* 对 Q1/Q2/Q3 matched pairs 产生不同预测；
* 不与 DSRF、DR-GS、RT-Splatting 的已有问题重复；
* 能在你当前服务器和时间条件下完成。

我的优先级是：

| 机制         | 可区分状态    | 难度 | 建议          |
| ---------- | -------- | -: | ----------- |
| 厚度/质量守恒消光  | Q1       |  低 | 仅作 baseline |
| 各向异性吸收或二色性 | Q3       |  中 | **最推荐**     |
| 方向相关散射     | Q2/Q3    | 中高 | 可作为扩展       |
| 应变诱导双折射    | Q3       |  高 | 有价值但需偏振     |
| 微结构历史/迟滞   | 超出当前 (F) | 很高 | 暂不做         |

RT-Splatting 已经表明静态半透明 Gaussian 中几何占据和光学不透明度应当区分，但它没有研究形变导致的材料光学状态演化；这仍给你的新问题留下空间。([arXiv][3])
