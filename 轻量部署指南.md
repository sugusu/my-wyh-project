# 轻量部署指南（TRGRGS + RecycleGS translab_full）

## 前置条件

- Python 3.10+, CUDA 环境
- 至少 8 GB 磁盘空间

## 步骤

### 1. 克隆代码

```bash
git clone https://github.com/sugusu/my-wyh-project.git
cd my-wyh-project
```

### 2. 安装 HF CLI 并登录

```bash
pip install huggingface-hub
hf auth login --token <你的 HF_TOKEN>
```

### 3. 下载所需数据

```bash
# TRGRGS 全部数据
hf download sugusu/my-pro-data TRGRGS/checkpoints --repo-type dataset --local-dir .
hf download sugusu/my-pro-data TRGRGS/outputs --repo-type dataset --local-dir .
hf download sugusu/my-pro-data TRGRGS/logs --repo-type dataset --local-dir .
hf download sugusu/my-pro-data TRGRGS/reports --repo-type dataset --local-dir .
hf download sugusu/my-pro-data TRGRGS/data/stage2s --repo-type dataset --local-dir .

# RecycleGS translab_full（TRGRGS 依赖的数据集）
hf download sugusu/my-pro RecycleGS/data/translab_full --repo-type model --local-dir .
```

### 4. 验证

```bash
echo "TRGRGS checkpoints: $(find TRGRGS/checkpoints -type f | wc -l)"
echo "TRGRGS outputs: $(find TRGRGS/outputs -type f | wc -l)"
echo "translab_full: $(find RecycleGS/data/translab_full -type f | wc -l)"
```

期望结果：
- TRGRGS checkpoints: 12844
- TRGRGS outputs: 571
- translab_full: 4735
