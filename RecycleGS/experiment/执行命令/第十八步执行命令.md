# 十、Divergence 判断树

## Case A：Step 0 已崩


根因范围：

```text
checkpoint restore
checkpoint schema
active_sh_degree
parameter internal representation
```

## Case B：Step 0 正常，Step 1 立即崩


```text

梯度爆炸
```

```text
gradient norm xyz
gradient norm features_dc
gradient norm opacity
gradient norm scaling
gradient norm rotation
```

---

## Case C：In-memory 正常，Save/Reload 崩

根因：

```text
PLY serialization
load_ply
SH degree
opacity transform
scale transform
```

## Case D：模型逐渐从22跌到8

输出首次：


对应 iteration。

## Case E：In-memory 和 reload 都正常，但 evaluate_recovery 是8

