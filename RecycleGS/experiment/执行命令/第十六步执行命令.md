## 停止条件

如果没有 iteration 15000 的训练 checkpoint：

```text
停止 Stage 2B。
```

不得：

```text
只从 point_cloud.ply 创建新的 optimizer，
然后声称是从15k恢复训练。
```

这种做法会丢失 Adam 的一阶、二阶动量，与 schedule control 不公平。

如果当前 checkpoint 只保存 Gaussian capture，但 TSGS 的 `restore()` 能够正确重建 optimizer state，则按照官方恢复流程执行。

# 十二、Stage 2B Gate

Mask-risk recovery 通过必须满足：

## 删除选择仍然更准

```text
mask-risk removed mean GT distance
>
random removed mean GT distance
```

## 渲染安全

两个场景均满足：

```text
mask-risk final PSNR >= random final PSNR - 0.01 dB
mask-risk final SSIM >= random final SSIM - 0.0005
mask-risk final LPIPS <= random final LPIPS + 0.001
```

## 几何安全

proxy 阶段：

```text
mask-risk SPR >= random SPR
mask-risk d_shift <= random d_shift
```

若有正式几何指标：

```text
Chamfer increase <= 2%
F-score drop <= 1 percentage point
```

## 数量固定

```text
N_final = N_baseline - K
```

训练过程中数量不得变化。

## 跨场景一致

scene_01 和 scene_03 均满足以上主要条件。

# 十三、Stage 2B 结论

只允许以下三种结论：

## PASS

```text
mask-risk 删除更准；
500步后渲染不差于 random；
几何 proxy 或正式指标不差于 random；
两个场景趋势一致；
Gaussian 数量固定。
```

下一步：

```text
进入 Stage 3：
检测几何欠覆盖区域并生成跨视角验证候选。
```

## CONDITIONAL

```text
一个场景通过，另一个场景仅轻微退化；
或 proxy 指标通过，但正式几何尚未完成。
```

下一步：

```text
补正式几何评价或加强边界/contribution保护；
不得进入完整重播种。
```

## FAIL

```text
mask-risk 500步后渲染或几何明显差于 random；
或数量发生变化；
或 optimizer-aware prune 失败。
```

下一步：

```text
停止重播种；
降低删除比例或加强保护机制。
```

# 十四、一键执行脚本

创建：

```text
scripts/stage2b_recovery_500.sh
```

内容：

```bash
#!/usr/bin/env bash
set -euo pipefail

cd /data/wyh/RecycleGS
export CUDA_VISIBLE_DEVICES=2,3
export PYTHONPATH=/data/wyh/RecycleGS:/data/wyh/repos/TSGS:${PYTHONPATH:-}

RECOVERY=configs/stage2/recovery_500_locked.yaml

echo "1/6 检查可恢复 checkpoint"
python3 src/prune/check_recovery_checkpoint.py \
  --scene01 configs/stage1/reliability_scene01.yaml \
  --scene03 configs/stage1/reliability_scene03.yaml

echo "2/6 审计 optimizer-aware prune"
python3 src/prune/audit_optimizer_aware_prune.py \
  --config configs/stage1/reliability_scene01.yaml \
  --recovery-config $RECOVERY

echo "3/6 运行20步 smoke test"
python3 src/prune/train_pruned_recovery.py \
  --scene-config configs/stage1/reliability_scene01.yaml \
  --recovery-config $RECOVERY \
  --method mask_risk \
  --removed-indices outputs/prune_only/scene_01/ratio_005/mask_risk/removed_indices.npy \
  --output-dir outputs/prune_only/scene_01/ratio_005/mask_risk/recovery_smoke \
  --override-steps 20 \
  --seed 0

echo "4/6 运行 scene_01 全部方法"
python3 src/prune/run_recovery_methods.py \
  --scene-config configs/stage1/reliability_scene01.yaml \
  --recovery-config $RECOVERY \
  --methods schedule_control random low_opacity low_contribution mask_risk \
  --seed 0

echo "5/6 运行 scene_03 全部方法"
python3 src/prune/run_recovery_methods.py \
  --scene-config configs/stage1/reliability_scene03.yaml \
  --recovery-config $RECOVERY \
  --methods schedule_control random low_opacity low_contribution mask_risk \
  --seed 0

echo "6/6 评价和公平性审计"
python3 src/evaluation/evaluate_recovery_500.py \
  --scene01 configs/stage1/reliability_scene01.yaml \
  --scene03 configs/stage1/reliability_scene03.yaml \
  --recovery-config $RECOVERY

python3 src/prune/audit_recovery_fairness.py \
  --scene01 configs/stage1/reliability_scene01.yaml \
  --scene03 configs/stage1/reliability_scene03.yaml \
  --recovery-config $RECOVERY

python3 src/prune/evaluate_stage2b_gate.py \
  --scene01 configs/stage1/reliability_scene01.yaml \
  --scene03 configs/stage1/reliability_scene03.yaml \
  --recovery-config $RECOVERY

echo "Stage 2B 完成。只有 PASS 才允许进入候选重播种。"
```
