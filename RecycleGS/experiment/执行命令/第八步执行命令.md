# 四、先验证 mesh 与相机是否真正对齐

不要再使用“全部 Gaussian bbox 中心”判断 mesh 对齐。

创建：

```text
src/evaluation/validate_mesh_camera_mask_alignment.py
```

功能：

1. 加载 GT mesh；
2. 均匀采样 100000 个 mesh surface points；
3. 使用 TSGS 原始相机 loader；
4. 均匀选择 32 个视角；
5. 将 mesh points 投影到这些视角；
6. 统计：

   * 投影在图像内比例；
   * 投影在透明 mask 内比例；
   * 每个视角 mask hit ratio；
   * 每个视角有效投影点数量；
7. 保存 8 个视角的：

   * RGB；
   * transparent mask；
   * mesh 投影散点；
   * mesh 投影与 mask overlay。

输出：

```text
outputs/debug/stage1c_scene01/mesh_camera_alignment.json
outputs/debug/stage1c_scene01/mesh_camera_alignment_report.md
outputs/debug/stage1c_scene01/mesh_projection/
```

判断规则：

```text
若 32 个视角的 median mask hit ratio >= 0.70：
    mesh 和相机坐标基本对齐。

若 median mask hit ratio < 0.40：
    坐标系、相机或 mesh 存在严重问题，停止。

若处于 0.40～0.70：
    人工查看 overlay 后再判断。
```

注意：

```text
不得使用 ICP 来提高 mask hit ratio。
```

运行：

```bash
cd /data/wyh/RecycleGS
export CUDA_VISIBLE_DEVICES=2,3
export PYTHONPATH=/data/wyh/RecycleGS:/data/wyh/repos/TSGS:${PYTHONPATH:-}

python3 src/evaluation/validate_mesh_camera_mask_alignment.py \
  --config configs/stage1/reliability_scene01.yaml
```

如果该检查失败，停止本阶段。

## 2. 近球形 Gaussian 没有稳定法线

定义 planarity confidence：

```text
s_min <= s_mid <= s_max

```

只有：

```text
planarity_confidence >= 0.20
```

时才计算 Gaussian normal conflict。

其余 Gaussian：

```text
normal_valid = false
```

# 十、重新计算局部表面支持

修改或创建：

```text
src/features/extract_object_surface_support.py
```

当前 `E_support` 强负相关，是因为背景平面通常局部密集，但离物体 mesh 很远。

修正要求：

1. 只对 object-supported Gaussian 建立 kNN；
2. `k=24`；
3. 邻域搜索不能包含 background 和 uncertain Gaussian；
4. 使用 object 域的局部距离尺度；
5. 对近球形 Gaussian，法线一致项降低权重；
6. 分别输出：

   * position density support；
   * normal alignment support；
   * combined support。

建议：

```text
S_position =
mean(exp(-distance² / (2 sigma²)))

S_normal =
mean(abs(dot(n_i, n_j)))，仅对有效法线邻居

S_combined =
S_position × (0.5 + 0.5 × S_normal)
```

风险：

```text
E_support_object = 1 - percentile_normalize(S_combined)
```

输出：

```text
outputs/reliability/scene_01/object_position_support.npy
outputs/reliability/scene_01/object_normal_support.npy
outputs/reliability/scene_01/object_surface_support_risk.npy
outputs/reliability/scene_01/object_surface_support_stats.json
```
