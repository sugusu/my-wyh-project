# 十五、一键分阶段脚本

创建：

```text
scripts/stage2bz_staged_recovery_validation.sh
```

内容：

```bash
#!/usr/bin/env bash
set -euo pipefail

cd /data/wyh/RecycleGS

export CUDA_VISIBLE_DEVICES=2,3
export PYTHONPATH=/data/wyh/RecycleGS:/data/wyh/repos/TSGS:${PYTHONPATH:-}

echo "1/7 Build unified evaluator identity"
python3 src/evaluation/build_evaluator_identity.py

echo "2/7 Zero-step evaluator equivalence"
python3 src/evaluation/test_zero_step_evaluator_equivalence.py \
  --config configs/stage1/reliability_scene01.yaml

echo "3/7 Audit old 17.62 PSNR semantics"
python3 src/evaluation/audit_divergence_psnr_source.py

echo "4/7 Run scene_01 schedule-control 500"
python3 src/prune/run_fixed_schedule_control.py \
  --scene-config configs/stage1/reliability_scene01.yaml \
  --recovery-config configs/stage2/recovery_500_locked.yaml \
  --steps 500

echo "5/7 Evaluate scene_01 mini gate"
python3 src/prune/run_fixed_recovery_pair.py \
  --scene-config configs/stage1/reliability_scene01.yaml \
  --recovery-config configs/stage2/recovery_500_locked.yaml \
  --methods random mask_risk

python3 src/prune/evaluate_scene01_recovery_mini_gate.py

echo "6/7 Run scene_03 staged recovery"
python3 src/prune/run_fixed_schedule_control.py \
  --scene-config configs/stage1/reliability_scene03.yaml \
  --recovery-config configs/stage2/recovery_500_locked.yaml \
  --steps 500

python3 src/prune/run_fixed_recovery_pair.py \
  --scene-config configs/stage1/reliability_scene03.yaml \
  --recovery-config configs/stage2/recovery_500_locked.yaml \
  --methods random mask_risk

echo "7/7 Final Stage 2B Gate v3"
python3 src/evaluation/evaluate_fixed_recovery_geometry_proxy.py

python3 src/prune/evaluate_stage2b_gate_v3.py

echo "Stage 2B-Z staged validation completed."
```

注意：

```text
脚本必须检查上一步 Gate。
```

例如：

```text
Zero-Step FAIL → exit 1
scene_01 schedule-control FAIL → exit 1
scene_01 Mini Gate FAIL → exit 1
scene_03 schedule-control FAIL → exit 1
```

不得无条件执行下一阶段。
