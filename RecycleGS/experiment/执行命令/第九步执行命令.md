# 七、重新定义 GT 几何误差表示

创建：

```text
src/evaluation/compute_geometry_error_labels_v2.py
```

仅用于离线评价。

对 candidate-object Gaussian 计算三类连续误差。

## 1. 中心距离

```text
d_center_norm =
center_to_mesh_distance / object_diameter
```

## 2. Scale 归一化距离

```text
d_scale =
center_to_mesh_distance / (normal_scale + eps)
```

## 3. Gaussian 表面代理距离

Gaussian 中心不一定必须落在 mesh 上，因此增加：

```text
d_surface_proxy_alpha1 =
max(0, center_to_mesh_distance - 1.0 * normal_scale)
d_surface_proxy_alpha2 =
max(0, center_to_mesh_distance - 2.0 * normal_scale)
```

归一化：

```text
d_surface_proxy_norm =
d_surface_proxy / object_diameter
```

注意：

* `alpha=1` 和 `alpha=2` 都报告；
* 不能根据哪个指标更好来选择 alpha；
* 这是标签敏感性分析，不参与风险计算。

输出：

```text
outputs/reliability/scene_01/candidate_geometry_errors_v2.npz
outputs/reliability/scene_01/candidate_geometry_error_stats.json
outputs/reliability/scene_01/candidate_geometry_error_report.md
```
