# 六、彻底重写 Surface Support v2

创建：

```text
src/features/extract_surface_support_v2.py
```

不要复用旧 `E_support_saved`。

输入：

```text
candidate_object_indices
Gaussian xyz
Gaussian scale
Gaussian normal
normal_valid
```

只在 candidate-object 内建立 kNN：

```text
k = 24
```

输出三个原始量：

## 1. Position support

```text
S_position =
mean(exp(-distance² / (2 sigma_i²)))
```

其中：

```text
sigma_i = candidate 邻域距离中位数
```

## 2. Normal support

只对法线有效的邻居计算：

```text
S_normal =
mean(abs(dot(n_i, n_j)))
```

如果有效法线邻居少于 3 个：

```text
normal_support_valid = false
```

不得填 0。

## 3. Combined support

```text
S_combined =
S_position × (0.5 + 0.5 × S_normal)
```

当 normal support 无效时：

```text
S_combined = S_position
```

使用固定函数进行分位数归一化：

```python
def percentile_normalize(x, low=5.0, high=95.0):
    lo = np.percentile(x[np.isfinite(x)], low)
    hi = np.percentile(x[np.isfinite(x)], high)
    y = np.clip((x - lo) / (hi - lo + 1e-8), 0, 1)
    return y, lo, hi
```

定义：

```text
support_confidence = percentile_normalize(S_combined)
E_support_v2 = 1 - support_confidence
```

必须立即验证：

```python
max_abs_difference = np.max(
    np.abs(E_support_v2 - (1.0 - support_confidence))
)

assert max_abs_difference < 1e-6
```

保存：

```text
candidate_global_indices.npy
support_position.npy
support_normal.npy
support_normal_valid.npy
support_combined.npy
support_confidence.npy
support_risk_v2.npy
support_v2_stats.json
```


# 十三、特征有效性规则

单个风险特征只有同时满足以下条件，才能称为“正向有效”：

```text
1. signed Spearman rho > 0.15；
2. bootstrap 95% CI 下界 > 0；
3. top10/bottom10 mean error ratio > 1.30；
4. 有效覆盖率 >= 30%；
5. 没有 NaN/Inf；
6. 特征定义方向符合非 GT 的物理解释。
```

若：

```text
rho < 0
```

则结论必须是：

```text
风险方向相反或该语义不适用于当前 candidate domain
```

不能对特征自动乘以 `-1`。
