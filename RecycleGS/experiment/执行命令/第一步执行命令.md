# 2. 定义实时日志函数

请在当前shell中执行：

````bash
cd /data/wyh/RecycleGS

log() {
  echo "" | tee -a /data/wyh/RecycleGS/realtime_log.md
  echo "### $(date '+%Y-%m-%d %H:%M:%S') $1" | tee -a /data/wyh/RecycleGS/realtime_log.md
}

run_cmd() {
  log "执行命令：$*"
  echo '```bash' >> /data/wyh/RecycleGS/realtime_log.md
  echo "$*" >> /data/wyh/RecycleGS/realtime_log.md
  echo '```' >> /data/wyh/RecycleGS/realtime_log.md
  echo "" >> /data/wyh/RecycleGS/realtime_log.md
  echo '```text' >> /data/wyh/RecycleGS/realtime_log.md
  eval "$*" 2>&1 | tee -a /data/wyh/RecycleGS/realtime_log.md
  echo '```' >> /data/wyh/RecycleGS/realtime_log.md
}
````

后续所有检查命令尽量通过 `run_cmd` 执行。
