## 10.1 不能简单从 PLY 重新初始化优化器

优先方式：

```text
加载原始 iteration 15000 training checkpoint
→ 在内存中按照 removed_indices 删除 Gaussian
→ 同步裁剪 optimizer state
→ 从 15000 训练到 15500
```

必须复用 TSGS/3DGS 内部的 optimizer-aware prune 方法。

需要同步裁剪：

```text
xyz
features_dc
features_rest
opacity
scaling
rotation
optimizer exp_avg
optimizer exp_avg_sq
densification statistics
max_radii2D
其他 per-Gaussian buffer
```

不得只删除 PLY 行后重新建立全新 optimizer，因为这会破坏方法间公平性。

如果没有可恢复的训练 checkpoint：

```text
停止 Stage 2B
先补齐可恢复 checkpoint
```

```text

7. 500 步中无 densification 或新增 Gaussian；
```

建议工程阈值：

```text
Chamfer increase <= 2%
F-score drop <= 1 percentage point
PSNR drop <= Random mean drop
```
