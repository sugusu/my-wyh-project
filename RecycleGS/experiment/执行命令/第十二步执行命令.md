# 九、Gate 1 判断规则

正式 Gate 1 通过要求：

```text
1. scene_01 和 scene_03 的 E_support_v2 均为正相关；
2. 两个场景 signed Spearman 均 > 0.15；
3. 两个场景 bootstrap 95% CI 下界均 > 0；
4. 两个场景 top10/bottom10 ratio 均 > 1.30；
5. 两个场景特征覆盖率均 >= 30%；
6. 两个场景 support 验证 max_abs_difference < 1e-5；
7. 全过程没有使用 GT 选择阈值或翻转特征方向。
```

如果全部满足：

```text
Gate 1 PASS
允许进入 Stage 2 prune-only
```

如果 scene_03 方向一致但强度略低：

```text
Gate 1 CONDITIONAL
继续分析 scene_07，不进入 prune
```

如果 scene_03 方向相反或接近零：

```text
Gate 1 FAIL
停止 prune，重新设计 reliability 特征
```

# 十、一键执行脚本

创建：

```text
scripts/stage1g_cross_scene_gate1.sh
```

内容：

```bash
#!/usr/bin/env bash
set -euo pipefail

cd /data/wyh/RecycleGS
export CUDA_VISIBLE_DEVICES=2,3
export PYTHONPATH=/data/wyh/RecycleGS:/data/wyh/repos/TSGS:${PYTHONPATH:-}

SCENE03_CONFIG=configs/stage1/reliability_scene03.yaml
LOCKED_CONFIG=configs/stage1/gate1_locked.yaml

echo "1/5 检查 scene_03 数据与坐标"
python3 src/evaluation/validate_mesh_camera_mask_alignment.py \
  --config $SCENE03_CONFIG

echo "2/5 建立 scene_03 candidate domain"
python3 src/reliability/build_checkpoint_candidate_domain.py \
  --config $SCENE03_CONFIG \
  --iteration 15000 \
  --locked-config $LOCKED_CONFIG

echo "3/5 提取 scene_03 Surface Support v2 和 Scale v2"
python3 src/features/extract_surface_support_v2.py \
  --config $SCENE03_CONFIG \
  --iteration 15000 \
  --locked-config $LOCKED_CONFIG

python3 src/features/extract_scale_anomaly_v2.py \
  --config $SCENE03_CONFIG \
  --iteration 15000 \
  --locked-config $LOCKED_CONFIG

echo "4/5 计算连续几何误差和有符号指标"
python3 src/evaluation/compute_checkpoint_geometry_errors.py \
  --config $SCENE03_CONFIG \
  --iteration 15000

python3 src/reliability/evaluate_signed_continuous_features.py \
  --config $SCENE03_CONFIG \
  --iteration 15000

echo "5/5 正式跨场景 Gate 1"
python3 src/reliability/evaluate_cross_scene_gate1.py \
  --scene01-config configs/stage1/reliability_scene01.yaml \
  --scene03-config configs/stage1/reliability_scene03.yaml \
  --locked-config $LOCKED_CONFIG

echo "Stage 1G 完成。只有 Gate 1 PASS 才允许进入 prune-only。"
```
