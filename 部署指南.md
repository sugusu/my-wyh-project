# 部署指南：在新服务器上还原实验环境

## 前置条件

- Python 3.10+
- CUDA 环境（用于 3D Gaussian Splatting）
- 至少 80 GB 可用磁盘空间

## 步骤

### 1. 安装依赖工具

```bash
# Git + Git LFS（用于下载大文件）
sudo apt install git git-lfs
git lfs install

# HuggingFace CLI（用于从 HF 下载数据）
pip install huggingface-hub

# 验证安装
git --version
hf --version
```

### 2. 克隆代码

```bash
git clone https://github.com/sugusu/my-wyh-project.git
cd my-wyh-project
```

### 3. 登录 HuggingFace 并下载数据

```bash
# 登录 HF（需要你的 HF token）
hf auth login --token <你的 HF_TOKEN>

# 下载所有大文件
bash download_data.sh
```

### 4. 验证文件完整性

下载完成后，检查关键目录是否存在且非空：

```bash
# 文件数检查
echo "=== Model repo ==="
echo -n "translab_full: "; find RecycleGS/data/translab_full -type f | wc -l
echo -n "baselines: "; find RecycleGS/baselines/tsgs_official_scene01_30k_v4 -type f | wc -l
echo -n "recovery: "; find RecycleGS/outputs/recovery -type f | wc -l
echo -n "prune_only: "; find RecycleGS/outputs/prune_only -type f | wc -l
echo -n "stage4_0_R2A_GT1: "; find DeformTransGS/experiments/stage4_0_R2A_GT1_gt_optical_semantics_closure -type f | wc -l
echo -n "stage5_0_R3_C2: "; find DeformTransGS/experiments/stage5_0_R3_C2_perspective_v2_validity -type f | wc -l

echo ""
echo "=== Dataset repo ==="
echo -n "stage4_0_attribute: "; find DeformTransGS/experiments/stage4_0_attribute_sufficiency_gate -type f | wc -l

echo ""
echo "=== Dataset repo (ReliablePeakGS) ==="
echo -n "data: "; find ReliablePeakGS/data -type f | wc -l
echo -n "external: "; find ReliablePeakGS/external -type f | wc -l
echo -n "environment: "; find ReliablePeakGS/environment -type f | wc -l

echo ""
echo "=== 大文件验证（应 > 1 MB）===
find RecycleGS/baselines -name "*.pth" -size +1M | head -3
find DeformTransGS/experiments -name "*.npy" -size +1M | head -3
```

### 5. 安装项目依赖

各项目自行安装依赖：

```bash
# DeformTransGS
cd DeformTransGS
pip install -r requirements.txt  # 如果存在
# 编译 CUDA 扩展
cd runtime && bash build.sh 2>/dev/null || python setup.py build_ext --inplace
cd ../..

# RecycleGS
cd RecycleGS
pip install -r requirements.txt 2>/dev/null
cd ..

# ReliablePeakGS
# (数据通过 download_data.sh 已下载，无额外 Python 依赖)
```
